SKYBOUND: GODFALL — v0.3.0-alpha "SkyLink Ignition"
Changes since v0.2.0-alpha
File: skybound_godfall_v0.3.0-alpha.html
========================================================

Pure feature addition on top of v0.2.0 — no systems removed, the
top-down combat core is untouched. This build adds the chain-kill
combo mechanic that later becomes the basis for the "Kickoff" theme
integration.

ADDED
- Chain-kill combo system: killing enemies in quick succession builds
  a chain (registerChainKill, considerChainTarget, chooseNextLink)
- Visual chain-target marking on screen (clearChainMarks)
- "Ignite" trigger for chain payoff (igniteSky)
- Kickoff relay wave-start mechanic (spawnKickoffRelay,
  spawnKickoffWave) — first appearance of the relay-destruction
  gate that becomes the theme's core mechanic in later builds

UNCHANGED FROM v0.2.0
- Top-down movement, bosses, radar, level select/unlock logic,
  fuel/shield/weather systems — no regressions, no removals

KNOWN LIMITATIONS
- Combo system has no milestone rewards yet (chain length is scored
  but doesn't grant items/power-ups)
- Still no wave structure within levels, no save system, no
  difficulty tiers, no missiles/weapon power-ups
