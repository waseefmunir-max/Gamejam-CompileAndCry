SKYBOUND
Version: v0.1.0-alpha
Release type: Initial alpha prototype

SUMMARY
The first playable build of SKYBOUND, establishing the core arcade flight-runner concept.

ADDED
- Single-file HTML5 Canvas game with no external runtime dependencies.
- Side-scrolling aircraft movement using Arrow keys or WASD.
- Ten-level campaign with progressively longer and harder missions.
- Three difficulty modes: Rookie, Veteran, and Extreme.
- Dynamic weather states: clear, sunset, storm, and night.
- Environmental hazards including towers and floating mines.
- Enemy types including turrets, patrol aircraft, and chasers.
- Fuel management, collectible coins, shield pickups, and boost pickups.
- Procedural sound effects, engine audio, and a small chiptune loop.
- Score tracking, level-complete screen, game-over screen, pause, mute, and campaign victory flow.
- Procedural visuals for sky, hills, clouds, weather, enemies, aircraft, particles, and HUD.

TECHNICAL NOTES
- Built entirely with vanilla HTML, CSS, JavaScript, and Canvas.
- All game logic and assets are contained or procedurally generated inside one HTML file.
- This build serves as the gameplay and technical foundation for later versions.

STATUS
- Early alpha.
- Core loop is playable, but progression and presentation are still experimental.
