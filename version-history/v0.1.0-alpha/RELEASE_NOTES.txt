SKYBOUND — v0.1.0-alpha
IUT ICT FEST 2026 GameJam
File: skybound_v0.1.0-alpha.html
========================================================

FIRST PLAYABLE BUILD.

Single-file HTML5 Canvas prototype (960x600). No external dependencies,
no assets — everything drawn/synthesized procedurally.

WHAT'S IN THIS BUILD
- Arcade flight-runner core loop: fly forward, dodge/collect, survive
- Player plane rendered and controlled with Arrow keys / WASD
- Procedural scrolling background (hills, clouds)
- Obstacle spawning (towers, mines) and pickups (fuel, power-ups)
- Basic collision + hit handling
- Fuel meter and shield mechanic
- Weather variation (cosmetic pass — clear/storm states)
- Level select screen (placeholder — single track of progression)
- Procedural sound via Web Audio (no external audio files)
- Pause (P) and mute (M) controls

KNOWN LIMITATIONS
- Side-view runner perspective — this is a prototype of the CONCEPT,
  not the final game direction
- No bosses, no waves, no theme-specific mechanics yet
- No save system — progress does not persist between sessions
- Single difficulty, no scoring depth

STATUS: Proof of concept only. This build establishes that a
dependency-free single-file Canvas game is viable within the jam's
"no additional software to install" web-build rules. Everything from
here is a redesign, not an iteration — see v0.2.0-alpha CHANGELOG for
the pivot to a top-down flight-combat format.
