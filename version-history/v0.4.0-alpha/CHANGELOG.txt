SKYBOUND: GODFALL — KICKOFF
Version: v0.4.0-alpha
Previous Version: v0.3.0-alpha
Release Type: Major Systems Update

============================================================
OVERVIEW
============================================================

SKYBOUND: GODFALL v0.4.0-alpha significantly expands the game
with structured combat waves, persistent progression, save systems,
new weapons, improved bosses, configurable controls, and a complete
KICKOFF-focused gameplay loop.

The SkyLink Ignition mechanic introduced in the previous version has
been expanded and renamed as the KICKOFF chain system.

This version moves the project from an experimental gameplay prototype
toward a more complete and replayable arcade campaign.

============================================================
ADDED
============================================================

WAVE SYSTEM
------------

- Added structured enemy waves for every level.

- Added different wave counts based on difficulty:

  - Rookie: 3 waves
  - Veteran: 4 waves
  - Extreme: 5 waves

- Added enemy-count scaling based on:

  - Current level
  - Current wave
  - Selected difficulty

- Added enemy health scaling across later waves and levels.

- Added wave-start and wave-complete notifications.

- Added wave restart support after player death.

- Added automatic progression from waves to boss encounters or level
  completion.

KICKOFF SYSTEM
--------------

- Renamed the SkyLink chain system to KICKOFF for stronger alignment
  with the GameJam theme.

- Added a relay that must be destroyed to kick off the assault.

- Added KICKOFF chain tracking.

- Added marked-target selection and energy-link effects.

- Added chain-based score multipliers.

- Added KICKOFF milestone rewards:

  - KICKOFF x5:
    Shield, additional missiles, and combat support.

  - KICKOFF x10:
    Afterburner boost, missiles, and a temporary weapon upgrade.

  - KICKOFF x15:
    Hull repair, missiles, and an extended weapon upgrade.

- Added a re-kickoff visual flare after defeating a boss.

WEAPONS AND COMBAT
------------------

- Added homing missiles.

- Added missile ammunition.

- Added missile pickups.

- Added missile explosion damage.

- Added temporary weapon upgrades:

  - Triple Shot
  - Spread Shot
  - Increased Damage

- Added weapon-upgrade timers.

- Added expanded support pickups:

  - Fuel

  - Shield
  - Boost
  - Missiles
  - Weapon upgrades

- Added improved enemy health and damage balancing.

- Added larger enemy pools for later levels and difficulties.

BOSSES

- Added unique boss names, colour palettes, and visual forms.

- Added multi-phase boss attack patterns.

- Added stronger final-boss scaling.

- Added dedicated boss warnings and phase-change notifications.

- Added boss-specific music mode support.

SAVE SYSTEM

- Added quick-save functionality.

- Added automatic saving.

- Added ten named manual save slots.

- Added save-slot loading.

CHANGED

- Replaced the previous timer-based enemy spawning system with structured

- Changed mission progression from distance-based survival to wave-based
  combat.

- Renamed SkyLink Ignition to the KICKOFF system.

- Reworked enemy spawning, health, movement, and difficulty scaling.

- Reworked the chain target system using persistent enemy identifiers.

- Reworked score calculation and milestone rewards.

- Reworked level progression and coin-unlock requirements.

- Reworked boss encounters with unique variants and attack phases.

- Reworked the player aircraft weapon system.

- Reworked pickup spawning and combat-support rewards.

- Reworked the game state to support saving and restoring complete runs.

- Reworked the menu, pause screen, settings, and result screens.

- Improved visual effects, explosions, particles, screen shake, and

- Improved audio handling with external files and procedural fallbacks.

============================================================
REMOVED
============================================================

- Removed the previous continuous, timer-based enemy-spawning structure.

- Removed the previous distance-goal progression system.

- Removed the SkyLink name from the primary chain mechanic.

- Removed session-only campaign progression.

- Replaced the earlier limited boss structure with a larger boss catalog.

============================================================
KNOWN LIMITATIONS
============================================================

- Game balance across all levels and difficulties is still experimental.

- Save files may become incompatible with major future changes.

- Some external artwork and audio assets may still be unavailable.

- Procedural fallback visuals remain placeholders.

- Enemy, missile, and boss collision behaviour may require refinement.

- Some menus and interfaces may require additional usability testing.

- God Mode access and cheated-run handling may require further testing.

- Performance may vary when a large number of enemies, projectiles, and
  visual effects appear simultaneously.

============================================================
TECHNICAL INFORMATION
============================================================

- Built using HTML5 Canvas, vanilla JavaScript, HTML, and CSS.

- Uses no required external libraries or runtime dependencies.

- Runs directly inside a modern web browser.

- All game systems remain contained inside a single HTML file.

- Uses browser local storage for:

  - Campaign progress
  - Settings
  - Quick saves
  - Autosaves
  - Manual save slots

- Save data uses a versioned serializable structure.

- Enemy and pickup objects use unique identifiers for restoring gameplay
  state and tracking chain targets.

- External image and audio files are optional.
